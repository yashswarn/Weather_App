# 🌦️ Weather App

A real-time weather app that fetches data from OpenWeatherMap API.

## 🚀 Features
- Search by city
- Display current weather info
- Clean, responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
